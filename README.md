# Agent-Forge

Plateforme d'orchestration d'agents IA multi-agents, isolée par microVM,
avec versioning Git intégré et dashboard web complet.

> Full implementation in the `release/phase1-phase2-complete` branch.

## Quick start

```bash
git clone https://github.com/neo-schobert/Agent-Forge.git
cd Agent-Forge
./install.sh
```
